# Privacy Policy

**Last Updated:** January 24, 2026

## Introduction

This Privacy Policy explains how the CMI Discord Bot ("the Bot", "we", "us", or "our") collects, uses, stores, and protects information from users. By using the Bot, you consent to the data practices described in this policy.

## 1. Information We Collect

### 1.1 Automatically Collected Information

When you use the Bot, we automatically collect:

**Server Information:**
- Discord server (guild) ID
- Server name (for logging purposes only)
- Server timezone settings
- Configured channel IDs
- Configured role IDs
- Custom nickname prefix settings

**User Information:**
- Discord user IDs (not usernames or display names)
- User timezone preferences
- Leadership permission assignments

**CMI Entry Data:**
- Leave dates and times
- Return dates and times
- Absence reasons (user-provided text)
- Timezone used for the entry
- Entry creation timestamp
- User ID of the person who created the entry

### 1.2 Information You Provide

You voluntarily provide:
- CMI absence reasons (optional text field)
- Timezone preferences
- Date and time information for absences

### 1.3 Information We Do NOT Collect

We do NOT collect:
- Message content from your server
- Direct messages
- Voice chat data
- User passwords or tokens
- Payment information (the Bot is free)
- IP addresses or location data beyond timezone settings
- Email addresses or phone numbers
- Personally identifiable information beyond Discord user IDs

## 2. How We Use Your Information

### 2.1 Primary Purposes

We use collected information to:
- Display CMI entries to appropriate users in your server
- Automatically assign/remove roles based on CMI status
- Add/remove nickname prefixes for users with active CMIs
- Generate daily reports (if enabled by server administrators)
- Provide CSV export functionality for leadership
- Enforce channel restrictions for CMI commands
- Store timezone preferences for accurate date/time display

### 2.2 Technical Purposes

We may use data for:
- Debugging errors and improving Bot functionality
- Monitoring Bot performance and uptime
- Preventing abuse or misuse of the Bot
- Complying with legal obligations

### 2.3 What We Do NOT Do

We do NOT:
- Sell, rent, or trade your data
- Share data with third parties for marketing purposes
- Use your data for advertising
- Train AI models on your data
- Monitor your conversations or activities outside Bot interactions

## 3. Data Storage and Security

### 3.1 Storage Location

All data is stored in:
- An SQLite database (`cmi.db`) on our hosting server
- Hourly backups retained for 7 days
- No data is stored in external databases or cloud services

### 3.2 Security Measures

We implement reasonable security measures:
- Database access restricted to the Bot application only
- Regular automated backups
- Server-level access controls
- No public exposure of database files

### 3.3 Data Retention

**CMI Entries:**
- Active entries: Retained indefinitely
- Completed entries: Automatically deleted after 90 days from return date
- This helps manage storage costs and protects user privacy

**Server Settings:**
- Retained while the Bot is in your server
- Deleted immediately when Bot is removed from server

**User Preferences:**
- Retained while user is in any server using the Bot
- Deleted when user leaves all servers using the Bot (via auto-cleanup)

### 3.4 Limitations

While we strive to protect your data, no method of electronic storage is 100% secure. We cannot guarantee absolute security.

## 4. Data Sharing and Disclosure

### 4.1 Within Your Server

Data is shared within your Discord server as follows:

**All Server Members:**
- Can see public CMI announcements (dates, reasons, status)

**Server Leadership (administrators and granted users):**
- Can view all CMI entries for the server
- Can create/edit/delete CMI entries for others
- Can export all CMI data via CSV
- Can view previous CMI history

**Individual Users:**
- Can view and manage their own CMI entries

### 4.2 External Sharing

We do NOT share data with third parties except:
- When required by law (e.g., valid legal process)
- To protect our rights, property, or safety
- With your explicit consent

We do NOT use third-party analytics, tracking, or advertising services.

### 4.3 Discord API

The Bot uses Discord's API to function. Discord's own Privacy Policy (https://discord.com/privacy) governs how Discord handles data transmitted through their platform.

## 5. Your Rights and Choices

### 5.1 Access Your Data

You can:
- View your own CMI entries via the Bot's interface
- Request leadership to export your data via CSV
- View timezone and preference settings you've configured

### 5.2 Modify Your Data

You can:
- Edit or delete your CMI entries at any time
- Update your timezone preferences
- Request leadership to modify entries created on your behalf

### 5.3 Delete Your Data

You can request data deletion by:
- Deleting your individual CMI entries via the Bot
- Asking server leadership to remove your entries
- Leaving the server (auto-cleanup will remove user-specific settings)
- Contacting us directly for complete removal

### 5.4 Opt-Out Options

- You can choose not to provide optional information (like CMI reasons)
- You can choose not to use the Bot at all
- Server administrators can remove the Bot entirely, deleting all server data

### 5.5 Data Portability

Server administrators can export all CMI data via the CSV export feature. This allows you to:
- Download a complete record of CMI entries
- Transfer data to other systems if needed
- Maintain your own backup copies

## 6. Children's Privacy
